# 使用常见问题

本页面收集 OpenClaw 使用过程中的常见问题及解决方案。

---

## 1. 消息发送失败

**问题**：发送消息没有响应

**解决方案**：
1. 检查 Gateway 是否运行
2. 确认 API Key 配置正确
3. 检查网络连接

---

## 2. 响应速度慢

**问题**：AI 回复很慢

**解决方案**：
1. 切换到更快的模型
2. 检查网络延迟
3. 减少上下文长度

---

## 3. 回复内容不完整

**问题**：AI 回复被截断

**解决方案**：
1. 增加 max_tokens 参数
2. 简化提示词
3. 分段提问

---

## 4. 技能无法使用

**问题**：安装的技能不能使用

**解决方案**：
1. 确认技能已安装：`openclaw skills list`
2. 检查技能配置
3. 重启 Gateway

---

## 5. 文件上传失败

**问题**：无法上传文件

**解决方案**：
1. 检查文件大小限制
2. 确认文件格式支持
3. 检查磁盘空间

---

## 6. 搜索不到结果

**问题**：memory 搜索没有结果

**解决方案**：
1. 执行索引：`openclaw memory index`
2. 确认文档在正确目录
3. 检查文档内容

---

## 7. 会话丢失

**问题**：重启后会话丢失

**解决方案**：
1. 配置会话持久化
2. 检查数据目录权限
3. 定期备份数据

---

## 8. 获取更多帮助

如果以上方法无法解决问题：
1. 搜索 [GitHub Issues](https://github.com/openclaw/openclaw/issues)
2. 加入 [Discord 社区](https://discord.com/invite/clawd)
