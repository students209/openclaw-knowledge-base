# 快速开始

欢迎使用 OpenClaw！本指南将帮助你在 5 分钟内完成安装并启动你的第一个 AI Agent。

---

## 系统要求

| 要求 | 说明 |
|------|------|
| Node.js | 18.x 或更高版本 |
| npm | 9.x 或更高版本 |
| 操作系统 | macOS / Windows / Linux |

---

## 安装方式

选择适合你的安装方式：

| 方式 | 适用场景 |
|------|----------|
| [npm 安装](./installation) | 推荐，最简单 |
| [Docker 安装](./docker) | 服务器部署 |
| [云端版本](./cloud) | 无需本地安装 |

---

## 快速安装

```bash
# 安装 OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version
```
