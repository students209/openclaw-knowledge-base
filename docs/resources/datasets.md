# 数据集

训练用数据集。

---

## 数据格式

| 格式 | 说明 |
|------|------|
| Markdown (.md) | 推荐格式，支持格式 |
| TXT (.txt) | 纯文本 |
| PDF (.pdf) | 需OCR处理 |
| JSON (.json) | 结构化数据 |

---

## 数据准备原则

1. **内容质量**：专业、准确
2. **结构清晰**：有层次、有目录
3. **格式规范**：Markdown 优先
4. **避免重复**：删除重复内容
