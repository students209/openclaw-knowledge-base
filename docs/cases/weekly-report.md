# 案例：周报生成器

> ⭐⭐⭐ 难度：进阶 | ⏱ 预计时间：1小时

---

## 效果演示

自动将工作记录整理成专业的周报：
- 接收工作输入
- 自动分类整理
- 输出 Markdown 格式

---

## 实现思路

1. 创建任务
2. 定义周报模板
3. 配置输出格式

---

## 核心配置

```json
{
  "name": "weekly-report",
  "systemPrompt": "你是一个周报生成助手。\n\n工作流程：\n1. 让用户输入本周工作内容\n2. 按格式整理成周报\n\n输出格式：\n```markdown\n# 周报 - [姓名] - [日期]\n\n## 本周工作概览\n[一句话总结]\n\n## 完成的工作\n- [项目1]: [具体内容]\n- [项目2]: [具体内容]\n\n## 遇到的问题\n- [问题描述]: [解决方案]\n\n## 下周计划\n- [计划1]\n- [计划2]\n```\n\n要求：\n- 条理清晰\n- 重点突出\n- 数据具体"
}
```

---

## 使用方式

```
用户1：本周完成了用户登录功能开发，修复了3个bug，优化了页面性能
用户2：还参加了产品需求评审会议

助手：
# 周报 - 张三 - 2024-01-15

## 本周工作概览
完成用户登录功能开发，优化页面性能，参与需求评审

## 完成的工作
- 用户登录功能：完成开发并上线
- Bug修复：修复3个线上问题
- 页面优化：提升加载速度30%
- 需求评审：参加产品需求评审会议

## 遇到的问题
- 无重大问题

## 下周计划
- 完成用户个人资料页面
- 开始开发消息通知功能
```

---

## 进阶：定时提醒

配合 cron 实现每周自动提醒：

```bash
# 每周五下午提醒写周报
cron add --schedule "0 17 * * 5" --task weekly-report --message "周末愉快！别忘了提交本周周报哦~"
```
