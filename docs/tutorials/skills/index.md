# 技能介绍

扩展 OpenClaw 能力的技能模块。

---

## 什么是技能

技能是 OpenClaw 的扩展模块，可以为 Agent 添加各种能力。

## 技能来源

| 来源 | 说明 |
|------|------|
| openclaw-bundled | 官方内置技能 |
| openclaw-extra | 官方扩展技能 |
| openclaw-workspace | 工作区技能 |
| ClawHub | 社区技能市场 |

## 技能状态

| 状态 | 说明 |
|------|------|
| ✓ ready | 已安装可用 |
| ✗ missing | 未安装 |

## 常用技能分类

- **效率工具**：天气、搜索、文档处理
- **开发工具**：代码执行、GitHub、PDF编辑
- **生活娱乐**：音乐、提醒、日历
- **AI/ML**：TTS、STT、图像生成
