# 云端版本

无需本地安装，直接使用 OpenClaw 云端版本。

---

## 什么是云端版本？

云端版本是 OpenClaw 提供的在线版本，无需在本地安装配置，通过浏览器即可直接使用。

---

## 云端版本特点

| 特点 | 说明 |
|------|------|
| 无需安装 | 打开浏览器就能用 |
| 无需配置 | 无需配置 Node.js/Docker |
| 自动更新 | 始终保持最新版本 |
| 数据安全 | 数据存储在云端 |
| 多端访问 | 电脑、手机都可以 |

## 适用场景

### 适合使用云端版本

- 不想自己部署服务器
- 追求简单快捷
- 移动办公需求
- 预算有限

### 适合本地部署

- 对数据安全性要求高
- 需要高度定制
- 有自己的服务器
- 离线使用需求
