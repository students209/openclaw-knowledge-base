# 基础配置

配置 OpenClaw 的 API Key 和模型。

---

## 配置 API Key

运行配置向导：

```bash
openclaw configure
```

按提示选择模型提供商并输入 API Key。

---

## 支持的模型

| 提供商 | 模型 |
|--------|------|
| OpenAI | GPT-4, GPT-3.5 |
| Anthropic | Claude 3 |
| 阿里云 | 通义千问 |
| Moonshot | Kimi |
| Google | Gemini |

---

## 配置文件位置

- macOS/Linux: `~/.openclaw/openclaw.json`
- Windows: `%USERPROFILE%\.openclaw\openclaw.json`
