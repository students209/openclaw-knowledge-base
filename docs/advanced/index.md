# 高级功能

OpenClaw 高级功能，帮助你更深入地使用 OpenClaw。

---

## 功能概览

| 功能 | 说明 | 难度 |
|------|------|------|
| [Prompt 工程](./prompt-engineering) | 优化提示词，获得更好的回答 | ⭐ |
| [API 使用](./api-usage) | 编程接口，集成到你的应用 | ⭐⭐ |
| [Webhook](./webhook) | 事件回调，自动化工作流 | ⭐⭐ |

---

## 什么是高级功能？

高级功能面向以下用户：
- 需要深度定制 OpenClaw
- 想要集成到现有系统
- 需要自动化工作流
- 开发自己的应用

---

## 快速导航

### Prompt 工程
- [提示词模板](../resources/prompts) - 常用提示词
- 技巧：角色设定、格式要求、分步指令

### API 使用
- HTTP API - 发送消息
- WebSocket - 实时对话
- 认证方式 - API Key / Token

### Webhook
- 事件类型 - message、agent、error
- 签名验证 - 确保请求安全
- 常见用途 - 消息转发、日志记录

| 场景 | 推荐功能 |

| 集成到网站 | API 使用 |
