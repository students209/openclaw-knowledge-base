# 安装验证

验证 OpenClaw 是否正确安装。

---

## 检查清单

### 1. 检查 CLI

```bash
openclaw --version
```

应该显示版本号。

### 2. 检查 Gateway

```bash
openclaw gateway status
```

### 3. 运行测试

```bash
openclaw config check
```
