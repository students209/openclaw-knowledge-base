# 案例：文件整理助手

> ⭐⭐ 难度：入门 | ⏱ 预计时间：30分钟

---

## 效果演示

自动整理文件：
- 按类型分类
- 按日期归档
- 重命名文件

---

## 实现思路

1. 扫描指定目录
2. 分析文件特征
3. 按规则整理

---

## 核心配置

```json
{
  "name": "file-organizer",
  "systemPrompt": "你是一个文件整理助手。\n\n功能：\n1. 按文件类型分类（图片/文档/视频等）\n2. 按日期整理\n3. 批量重命名\n4. 清理重复文件\n\n整理规则：\n- 图片：Images/年月/\n- 文档：Documents/类型/\n- 下载：Downloads/年月/",
  "skills": ["nano-pdf"]
}
```

---

## 使用方式

```
用户：帮我整理下载文件夹
AI：（扫描文件夹，分析文件，整理到对应目录）

用户：帮我把照片按日期整理
AI：（读取照片EXIF信息，按拍摄日期移动到对应文件夹）
```

---

## 进阶：自动整理

配合 cron 实现定时自动整理！

```bash
# 每周日自动整理下载文件夹
cron add --schedule "0 22 * * 0" --task file-organizer --message "整理下载文件夹"
```
