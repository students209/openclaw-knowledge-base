# 基础教程

学习 OpenClaw 的核心功能，从入门到进阶。

---

## 教程目录

### 基础篇

| 教程 | 说明 | 难度 | 时间 |
|------|------|------|------|
| [首次启动](./basic/first-start) | 启动你的第一个 Agent | ⭐ | 5 分钟 |
| [基础配置](./basic/config) | 配置 API Key 和模型 | ⭐ | 10 分钟 |
| [界面介绍](./basic/interface) | 了解 OpenClaw 界面 | ⭐ | 5 分钟 |
| [第一个任务](./basic/first-task) | 完成你的第一个任务 | ⭐ | 10 分钟 |

### 进阶篇

| 教程 | 说明 | 难度 |
|------|------|------|
| [技能市场](./skills/) | 扩展 Agent 能力 | ⭐⭐ |
| [训练指南](./training/) | 让 AI 学习你的知识 | ⭐⭐⭐ |

## 学习路径建议

```
新手入门（基础篇）
    ↓
安装配置 → 发送第一条消息 → 完成第一个任务
    ↓
进阶学习（进阶篇）
    ↓
安装技能 → 训练知识库 → 高级功能
```
