## 项目 1：自动回复机器人

### 目标

创建一个 Discord/Telegram 机器人，能够：
- 自动回复用户消息
- 支持关键词匹配
- 支持智能回答

### 步骤

#### 1. 创建任务

```bash
openclaw task create auto-reply-bot
```

#### 2. 配置系统提示词

```json
{
  "systemPrompt": "你是一个自动回复机器人，负责回复用户的消息。\n\n回复规则：\n1. 如果用户问候，热情回应\n2. 如果用户询问产品功能，用简洁易懂的语言介绍\n3. 如果用户遇到问题，提供解决方案\n4. 如果不知道答案，礼貌地说明并建议用户联系人工客服\n\n注意：保持友好、专业的态度。"
}
```

#### 3. 接入 Discord（参考 Day 5）

```json
{
  "channels": {
    "discord": {
      "enabled": true,
      "token": "YOUR_TOKEN",
      "task": "auto-reply-bot"
    }
  }
}
```

#### 4. 测试

在 Discord 中发送消息，机器人会自动回复！

## 项目 2：周报生成器

### 目标

创建一个能够：
- 接收工作记录
- 自动整理成周报格式
- 输出专业的周报

### 步骤

#### 1. 创建任务

```bash
openclaw task create weekly-report
```

#### 2. 配置系统提示词

```json
{
  "systemPrompt": "你是一个周报生成助手。\n\n工作流程：\n1. 让用户输入本周完成的工作内容\n2. 按以下格式整理：\n   - 本周工作概览\n   - 完成的工作（按项目分类）\n   - 遇到的问题和解决方案\n   - 下周计划\n\n3. 输出格式要求：\n   - 使用 Markdown 格式\n   - 条理清晰\n   - 重点突出"
}
```

#### 3. 使用方式

```
用户：本周完成了用户登录功能开发，修复了3个bug，优化了页面性能
助手：（自动整理成周报格式）
```

## 项目 3：客服助手（进阶）

### 目标

创建一个专业的客服机器人：
- 了解产品知识（需要训练）
- 自动回答常见问题
- 复杂问题转人工

### 步骤

#### 1. 准备知识库

创建产品FAQ文档：

```markdown
# 产品FAQ

## 产品功能
Q: 产品支持哪些平台？
A: 支持 Web、iOS、Android 三个平台。

## 定价
Q: 产品的价格是多少？
A: 免费版基础功能，付费版 ¥99/月起。

## 技术支持
Q: 如何联系客服？
A: 发送邮件至 support@example.com

... (更多 FAQ)
```

#### 2. 训练知识库

```bash
# 将文档放入训练目录
mkdir -p ~/.openclaw/training
cp product-faq.md ~/.openclaw/training/

# 执行训练
openclaw train
```

#### 3. 配置任务

```json
{
  "name": "customer-service",
  "systemPrompt": "你是我们的产品客服，熟悉产品FAQ中的所有内容。\n\n回复规则：\n1. 优先从知识库中查找答案\n2. 直接引用知识库内容回答\n3. 如果知识库没有，引导用户联系人工客服\n\n记住：保持专业、友好的态度。",
  "skills": []
}
```

#### 4. 接入平台

按照 Day 5 的步骤接入 Discord 或其他平台。

## 进阶：组合多个技能

可以让机器人使用多个技能：

```json
{
  "skills": ["weather", "summarize", "tavily"]
}
```

实现更强大的功能：
- 查询天气
- 总结文档
- 搜索信息
