# Windows 安装

本指南帮助你在 Windows 系统上安装 OpenClaw。

---

## 系统要求

- Windows 10 或更高版本
- Node.js 18.x 或更高版本

---

## 安装步骤

### 1. 安装 Node.js

访问 [nodejs.org](https://nodejs.org/) 下载并安装 LTS 版本。

### 2. 安装 OpenClaw

打开 PowerShell 或 CMD：

```bash
npm install -g openclaw
```

### 3. 验证安装

```bash
openclaw --version
```
