# 案例：内容创作助手

> ⭐ 难度：入门 | ⏱ 预计时间：45分钟

---

## 效果演示

自动生成各类内容：
- 文章
- 标题
- 摘要
- 社交媒体文案

---

## 实现思路

1. 创建任务
2. 配置创作提示词
3. 使用技能增强

---

## 核心配置

```json
{
  "name": "content-creator",
  "systemPrompt": "你是一个内容创作助手，擅长各种类型的文案创作。\n\n创作类型：\n1. 文章/博客\n2. 标题\n3. 摘要\n4. 社交媒体文案\n5. 产品描述\n\n风格：\n- 语言生动有趣\n- 结构清晰\n- 根据需求调整",
  "skills": ["summarize"]
}
```

---

## 使用示例

### 生成文章

```
用户：帮我写一篇关于 AI Agent 的科普文章，500字左右
AI：（生成文章...）
```

### 生成标题

```
用户：给这篇文章起5个标题
[文章内容]
AI：1. xxx
    2. xxx
    ...
```

### 生成社交媒体文案

```
用户：帮我写一条关于产品上线的小红书文案
AI：（生成文案...）
```
