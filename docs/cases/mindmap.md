# 案例：思维导图助手

> ⭐⭐⭐ 难度：进阶 | ⏱ 预计时间：1小时

---

## 效果演示

自动生成思维导图：
- 文章要点总结
- 书籍结构梳理
- 知识体系构建

---

## 实现思路

1. 输入主题或文档
2. AI 分析结构
3. 生成 Markdown 格式
4. 导入思维导图工具

---

## 核心配置

```json
{
  "name": "mindmap-generator",
  "systemPrompt": "你是一个思维导图生成助手。\n\n功能：\n1. 阅读输入内容\n2. 提取关键主题和子主题\n3. 构建层级结构\n4. 输出 Markdown 格式\n\n输出格式示例：\n# 主题\n## 一级主题\n### 二级主题\n- 要点1\n- 要点2\n\n支持的工具：XMind, MindNode, 幕布等"
}
```

---

## 使用方式

### 从文章生成

```
用户：帮我把这篇文章生成思维导图
[粘贴文章内容]
AI：（生成思维导图结构...）
```

### 从主题生成

```
用户：帮我生成"AI Agent 技术架构"的思维导图
AI：（生成完整的技术架构导图...）
```

---
