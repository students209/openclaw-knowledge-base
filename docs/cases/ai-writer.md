# 案例：AI写作助手

> ⭐⭐ 难度：入门 | ⏱ 预计时间：30分钟

---

## 效果演示

自动生成各类文章：
- 博客文章
- 社交媒体文案
- 产品描述

---

## 实现思路

1. 配置写作提示词
2. 提供主题和要求
3. AI 生成内容

---

## 核心配置

```json
{
  "name": "ai-writer",
  "systemPrompt": "你是一个专业的内容写作者。\n\n擅长类型：\n- 博客文章\n- 社交媒体文案\n- 产品描述\n- 新闻稿\n\n风格：\n- 语言生动流畅\n- 结构清晰\n- 根据受众调整风格"
}
```

---

## 使用示例

### 博客文章

```
用户：帮我写一篇关于 AI Agent 的博客文章，1000字左右
AI：（生成完整博客文章...）
```

### 社交媒体

```
用户：帮我写一条关于产品上线的小红书文案
AI：（生成吸引人的文案...）
```

---

## 进阶：搭配技能

```bash
# 安装相关技能
openclaw skills install summarize  # 摘要
openclaw skills install translate  # 翻译
```

实现：
- 写完文章 → 自动生成摘要
- 中文文章 → 自动翻译英文
