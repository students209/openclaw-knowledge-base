# 训练介绍

让 Agent 学习你的专业知识。

---

## 什么是训练

通过上传文档让 AI 学习特定领域的知识，训练后 Agent 可以：
- 回答你专属领域的问题
- 使用你的文档内容作为参考
- 提供更专业的回答

## 为什么需要训练

- 通用 AI 不了解你的具体需求
- 训练后可以引用你的文档
- 提供更准确、更专业的回答

---

## 训练方式

OpenClaw 提供两种训练方式：

| 方式 | 说明 |
|------|------|
| 文档训练 | 上传文档，让 AI 学习文档内容 |
| 记忆训练 | 使用 memory 命令管理知识库 |

---

## 快速开始

```bash
# 查看记忆状态
openclaw memory status

# 重新索引
openclaw memory index

# 搜索内容
openclaw memory search <关键词>
```
