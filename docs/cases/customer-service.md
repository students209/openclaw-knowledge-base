# 案例：客服助手

> ⭐⭐⭐⭐ 难度：高级 | ⏱ 预计时间：2小时

---

## 效果演示

智能客服机器人：
- 7×24 小时在线
- 自动回答常见问题
- 复杂问题转人工

![客服助手效果](./images/customer-service-demo.gif)

---

## 实现思路

1. **准备知识库**：整理产品 FAQ 文档
2. **训练 AI**：让 AI 学习知识库
3. **配置回复规则**：定义回答逻辑
4. **接入平台**：连接 Discord/飞书等

---

## 步骤 1：准备知识库

创建产品 FAQ 文档 `training/faq.md`：

```markdown
# 产品常见问题

## 产品功能
Q: 产品支持哪些平台？
A: 支持 Web、iOS、Android 三个平台。

Q: 有免费版吗？
A: 有的，免费版提供基础功能。

## 定价
Q: 付费版价格？
A: 个人版 ¥29/月，团队版 ¥99/月。

## 技术支持
Q: 如何联系客服？
A: 发送邮件至 support@example.com 或加入微信群。

## 退款
Q: 支持退款吗？
A: 7天内无条件退款。
```

## 步骤 2：训练知识库

```bash
# 创建训练目录
mkdir -p ~/.openclaw/training
cp faq.md ~/.openclaw/training/

# 执行训练
openclaw train
```

## 步骤 3：配置任务

```json
{
  "name": "customer-service",
  "systemPrompt": "你是一个专业的产品客服，熟悉产品知识库。\n\n回复规则：\n1. 优先从知识库中查找答案\n2. 直接、简洁地回答问题\n3. 如果知识库没有，礼貌说明\n4. 适当引导用户了解更多\n\n注意：保持友好、专业。",
  "skills": []
}
```

## 步骤 4：接入平台

参考 Day 5 教程接入 Discord 或飞书。

---

## 进阶：转人工机制

在提示词中添加转人工规则：

```
如果用户询问以下问题，引导转人工：
- 复杂的技术问题
- 投诉建议
- 商务合作

转人工话术：
"这个问题我需要转接人工客服处理，请稍等..."
```

## 常见问题

### 1. AI 不回答知识库内容

- 确认训练已执行
- 检查知识库文档格式
- 调整提示词强调使用知识库

### 2. 回答不够准确

- 优化知识库内容
- 增加更多 FAQ
- 调整训练参数
