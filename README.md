# OpenClaw 知识库

OpenClaw 中文社区官方文档站

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```

## 项目结构

```
docs/
├── .vitepress/     # VitePress 配置
├── getting-started/ # 快速开始
├── tutorials/      # 教程
├── advanced/       # 高级功能
├── deployment/     # 部署方案
├── channels/       # 频道集成
├── resources/      # 资源中心
├── faq/            # 常见问题
├── videos/         # 视频教程
└── about/          # 关于
```
