# 数据准备

准备训练数据。

---

## 支持的格式

| 格式 | 说明 | 推荐程度 |
|------|------|----------|
| Markdown (.md) | 最佳选择，支持格式 | ⭐⭐⭐ |
| PDF (.pdf) | 支持，但需 OCR | ⭐⭐ |
| TXT (.txt) | 纯文本支持 | ⭐⭐ |
| Word (.docx) | 部分支持 | ⭐ |

---

## 数据要求

### 文件大小
- 单个文件建议不超过 10MB
- 总知识库建议不超过 100MB

### 内容要求
- 内容清晰、结构化
- 避免大量重复内容
- 使用规范的术语

---

## 推荐的文档结构

### Markdown 格式示例

```markdown
# 文档标题

## 第一章

内容介绍...

### 小节

详细说明...
```

### 建议包含

- 清晰的标题和目录
- 章节分段
- 关键术语解释
- 示例代码或案例

---

## 数据存放位置

| 系统 | 位置 |
|------|------|
| macOS/Linux | `~/.openclaw/memory/` |
| Windows | `%USERPROFILE%\.openclaw\memory\` |

---

## 准备步骤

1. 整理文档内容
2. 转换为 Markdown 格式
3. 检查内容质量
4. 放入知识库目录
5. 执行索引命令
