# 贡献指南

如何为 OpenClaw 贡献代码。

---

## 贡献方式

| 方式 | 说明 |
|------|------|
| 代码贡献 | 提交 PR |
| 问题反馈 | 提交 Issue |
| 文档完善 | 完善文档 |
| 社区支持 | 回答问题 |

---

## 开发环境

```bash
# 克隆仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install

# 开发模式
npm run dev
```

---

## 提交 Issue

1. 搜索是否已有类似问题
2. 使用 Issue 模板
3. 提供复现步骤

---

## 提交 PR

1. Fork 仓库
2. 创建分支
3. 提交更改
4. 提交 PR

---

## 代码规范

- 遵循 ESLint 规则
- 编写测试用例
- 更新文档
