# 配置常见问题

本页面收集 OpenClaw 配置过程中的常见问题及解决方案。

---

## 1. API Key 无效

**问题**：配置 API Key 后提示无效

**解决方案**：
1. 检查 API Key 是否正确复制
2. 确认 API Key 是否过期
3. 确认API Key 有足够额度

---

## 2. 模型切换失败

**问题**：无法切换到其他模型

**解决方案**：
1. 确认已配置对应模型的 API Key
2. 检查模型名称是否正确
3. 查看模型是否可用

---

## 3. 配置文件不生效

**问题**：修改配置后没有生效

**解决方案**：
1. 重启 Gateway：`openclaw gateway restart`
2. 检查 JSON 格式是否正确
3. 确认配置文件路径正确

---

## 4. 环境变量不生效

**问题**：设置了环境变量但没有生效

**解决方案**：
1. 确保环境变量在启动前设置
2. 重启终端或服务
3. 使用 `openclaw config get` 验证

---

## 5. 频道配置失败

**问题**：Discord/Telegram 等频道配置失败

**解决方案**：
1. 确认 Token 正确
2. 检查 Bot 权限
3. 查看频道是否正确添加

---

## 6. 端口被占用

**问题**：启动服务提示端口被占用

**解决方案**：
1. 使用其他端口：`--port 3001`
2. 停止占用端口的进程
3. 查看 `lsof -i :3000`

---

## 7. 内存不足

**问题**：运行提示内存不足

**解决方案**：
1. 增加服务器内存
2. 关闭不必要的程序
3. 使用轻量级模型

---

## 8. 获取更多帮助
