# Prompt 工程

优化你的提示词，获得更好的回答。

---

## 最佳实践

### 1. 清晰的指令

```bash
# ❌ 模糊的指令
写点什么

# ✅ 清晰的指令
写一篇关于AI助手的文章，要求：
- 500字左右
- 面向初学者
- 包含实际应用场景
```

### 2. 提供上下文

```bash
# ❌ 缺乏上下文
怎么配置？

# ✅ 提供上下文
我在使用OpenClaw，配置API Key时报错，如何解决？
```

### 3. 给出示例

```bash
# ❌ 没有示例
帮我写邮件

# ✅ 给出示例
帮我写一封求职邮件，格式如下：
标题：XXX
开头：自我介绍
正文：为什么想做这份工作
结尾：期待面试机会
```

## 常用技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 分步指令 | 复杂任务拆分成步骤 | 第一步...第二步... |
| 角色设定 | 指定AI扮演的角色 | 你是一位资深产品经理 |
| 格式要求 | 指定输出格式 | 用表格呈现、以JSON格式输出 |
| 限制条件 | 限制回答范围 | 只列出免费工具、字数限制 |

## 常用提示词模板

### 1. 代码审查

```
你是一位资深代码审查员，请审查以下代码：
1. 找出潜在的bug
2. 指出安全问题
3. 建议优化方案
```

### 2. 内容总结

```
请总结以下内容：
- 用一句话概括核心观点
- 列出3个关键要点
- 用通俗易懂的语言解释
```

### 3. 问题解答

```
我遇到的问题是：[具体问题]
我的环境是：[环境信息]
请提供详细的解决步骤
```
