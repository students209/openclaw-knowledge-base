# 第一个任务

完成你的第一个 OpenClaw 任务。

---

## 发送第一条消息

配置完成后，在 Web 界面或连接的频道中发送消息：

```
你好，请帮我写一个简单的 Python 函数，计算斐波那契数列
```

---

## 常用任务示例

### 1. 代码生成

```
帮我写一个 Python 函数，实现二分查找
```

### 2. 文档翻译

```
把这篇英文文章翻译成中文
```

### 3. 数据分析

```
分析这个 CSV 文件中的销售数据，找出增长最快的月份
```

### 4. 文案撰写

```
帮我写一篇关于 AI 助手的小红书种草文案
```

### 5. 知识问答

```
OpenClaw 支持哪些模型？
```

---

## 高级功能

### 使用技能

安装技能后，可以指定技能执行任务：

```
@技能名称 任务内容
```

### 文件处理

```
帮我分析这个 PDF 文件的内容
```
