# 案例：爬虫助手

> ⭐⭐⭐ 难度：进阶 | ⏱ 预计时间：1.5小时

---

## 效果演示

自动化数据采集：
- 网页内容抓取
- 数据清洗
- 结构化存储

---

## 实现思路

1. 配置爬取规则
2. 执行爬取任务
3. 数据清洗存储

---

## 核心配置

```json
{
  "name": "crawler-bot",
  "systemPrompt": "你是一个爬虫助手。\n\n功能：\n1. 根据URL抓取网页内容\n2. 提取关键信息\n3. 清洗数据\n4. 输出为JSON/CSV\n\n注意：\n- 遵守 robots.txt 规则\n- 控制爬取频率\n- 仅用于合法用途"
}
```

---

## 使用方式

### 简单爬取

```
用户：帮我爬取 https://example.com 的标题和链接
AI：（抓取并提取...）
```

### 数据采集

```
用户：帮我采集科技新闻网站的最新新闻，提取标题、链接、发布时间
AI：（抓取、提取、清洗...）
```

---

## 进阶：定时采集

配合 cron 实现定时数据更新！

```bash
# 每天早上8点采集行业资讯
cron add --schedule "0 8 * * *" --task crawler-bot --message "采集最新资讯"
```

---

## 注意事项

1. **遵守规则** - 遵守网站 robots.txt
2. **合理频率** - 避免对目标网站造成压力
3. **合法用途** - 仅用于正当需求
