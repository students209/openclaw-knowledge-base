# 训练概念

了解 OpenClaw 的训练机制。

---

## 核心概念

| 概念 | 说明 |
|------|------|
| 知识库 | 存储训练文档的地方 |
| 文档索引 | 将文档转换为向量，便于检索 |
| 向量检索 | 找到与问题最相关的文档片段 |
| Embedding | 将文本转换为数学向量的技术 |

---

## 工作原理

```
用户问题 → 向量化 → 检索相似文档 → 结合文档生成回答
```

1. **上传文档**：将 Markdown、PDF、TXT 文件上传到知识库
2. **自动索引**：系统自动将文档转换为向量并存储
3. **检索回答**：当用户提问时，检索相关文档片段
4. **生成回答**：结合检索到的内容生成回答

---

## 支持的功能

- 文档自动分块
- 向量相似度检索
- 多文档关联
- 实时更新索引

---

## 相关命令

```bash
# 查看记忆状态
openclaw memory status

# 查看详细状态
openclaw memory status --deep

# 重新索引
openclaw memory index

# 强制重新索引
openclaw memory index --force

# 搜索
openclaw memory search <关键词>
```
