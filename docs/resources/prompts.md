# 提示词模板

常用提示词模板。

---

## 写作助手

### 1. 文章撰写

```
请帮我写一篇关于[主题]的文章，要求：
- 字数：约[数字]字
- 风格：[正式/轻松/专业]
- 目标读者：[描述]
- 结构：开头→[要点]→结尾
```

### 2. 社交媒体

```
帮我写一篇[平台]帖子，主题是[主题]：
- 长度：[短/中/长]
- 风格：[种草/干货/故事]
- 包含[话题标签]
```

### 3. 邮件

```
写一封[类型]邮件：
- 收件人：[角色]
- 目的：[目的]
- 语气：[正式/友好]
```

---

## 代码助手

### 1. 代码审查

```
请审查以下代码：
1. 找出潜在bug
2. 指出安全问题
3. 建议优化方案
```

### 2. 代码生成

```
用[语言]实现[功能]：
- 输入：[描述]
- 输出：[描述]
- 要求：[性能/可读性等]
```

---

## 学习助手

### 1. 概念解释

```
用通俗易懂的语言解释[概念]：
- 比喻/例子
- 核心要点（3点）
- 适用场景
```

### 2. 学习总结

```
总结[主题]的核心内容：
- 一句话概括
- 关键概念
- 实践建议
```
