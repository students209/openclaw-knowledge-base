# macOS 安装

本指南帮助你在 macOS 系统上安装 OpenClaw。

---

## 系统要求

- macOS 10.15 或更高版本
- Node.js 18.x 或更高版本

---

## 安装步骤

### 1. 安装 Node.js

**方式一：使用 Homebrew**

```bash
brew install node
```

**方式二：官网下载**

访问 [nodejs.org](https://nodejs.org/) 下载并安装 LTS 版本。

### 2. 安装 OpenClaw

```bash
npm install -g openclaw
```

### 3. 验证安装

```bash
openclaw --version
```
