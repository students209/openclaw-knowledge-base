# 部署方案

生产环境部署指南。

---

## 部署选项

| 方式 | 难度 | 适合场景 |
|------|------|----------|
| [Docker](./docker) | ⭐ | 快速部署、容器化 |
| [服务器](./server) | ⭐⭐ | 传统服务器部署 |
| [群晖 NAS](./nas) | ⭐⭐ | NAS 用户 |
| [Render](./render) | ⭐ | 免费云平台 |
| [Railway](./railway) | ⭐ | 免费云平台 |
| [域名配置](./domain) | ⭐⭐ | 自定义域名+HTTPS |

---

## 选择建议

| 场景 | 推荐方案 |
|------|----------|
| 个人使用 | Docker / 本地 |
| 快速上线 | Render / Railway |
| 企业使用 | 服务器 + Docker |
| NAS 用户 | 群晖 Docker |

---

## 部署前准备

1. 确认 Node.js 22.12.0+
2. 准备域名（可选）
3. 选择服务器/平台

---

选择适合你的部署方式开始部署。
